# 🎓 Student Placement Predictor

A Machine Learning web application that predicts whether a student is likely to be placed based on academic performance and skill parameters.

---

## 🚀 Features
- 📊 Predict placement using ML model
- 🎯 Interactive UI with Streamlit
- 📈 Data visualization with graphs
- ⚡ Real-time user input predictions
- ✅ ~90% accuracy on realistic dataset

---

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit
- Matplotlib

---

## 📊 Model Details
- Algorithm: Logistic Regression
- Evaluation: Train-Test Split
- Accuracy: ~90% (after introducing realistic variability)
